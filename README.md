# File_creator
文件/文件夹多样化创建器

## 项目文件说明

### File_creator
基础版文件/文件夹创建工具，提供简单的交互式界面，支持基本的文件和文件夹批量创建功能。

### File_creator-v2  
增强版文件/文件夹创建工具，在基础版上增加了以下功能：
- 无限数量创建（无文件数量限制）
- API接口支持（命令行参数调用）
- 智能文件名模板系统
- 完整的权限管理
- 远程文件操作（HTTP/HTTPS/FTP）
- 自动脚本复制功能

### manual.sh
交互式使用手册脚本，提供多种查看方式：
- 在浏览器中打开完整HTML手册
- 命令行快速参考
- 系统依赖检查
- 工具使用指导

### enhanced_file_creator_manual.html
自动生成的HTML格式完整手册，包含：
- 详细的工具介绍和特性说明
- 交互模式和API模式使用指南
- 文件名模板系统说明
- 权限管理指南
- 远程操作教程
- 实际使用示例和故障排除

这些工具共同构成了一个完整的文件/文件夹管理解决方案，满足从简单到复杂的各种创建需求。
